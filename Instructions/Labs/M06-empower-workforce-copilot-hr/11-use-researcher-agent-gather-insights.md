# 演習 2、タスク 4:リサーチ ツール エージェントを使用して従業員の転勤に関する分析情報を収集する
---
あなたは、ノース ダコタ州ファーゴの Adatum オフィスへの転勤の一環として、ファーゴおよびノースダコタ州とミネソタ州の両方の近隣のコミュニティに住むのがどのようなものかを調べたいと考えています。 この情報を使用して、家族が住む場所を決定する参考にしたいと考えています。 重要な考慮事項には、生活の質、住宅の選択肢、通勤時間、レクリエーションの機会、コミュニティの施設、子供のための優れた教育プログラムへのアクセスなどがあります。

次に、これらのコミュニティがある都市を分析し、これらの要素の組み合わせに基づいて考慮すべき上位のエリアのランキングを提供するようにリサーチ ツールに依頼します。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーでは、引き続き **HR セルフサービス アシスタント** エージェントが表示されている必要があります。 ナビゲーション ウィンドウのエージェントの一覧で、**[リサーチ ツール]** エージェントを選択します。

2.  エージェントがこのクエリに対して **[Web]** ソースのみを使用するように指定します。

3.  ノースダコタ州とミネソタ州の両方で、ファーゴの周辺の半径 40 マイル以内にあるすべてのコミュニティを分析するレポートを作成するように、エージェントに求めるプロンプトを入力します。 使用可能な最新のデータを使用します。 全体的な生活の質と家族向けかどうかに基づいて都市をランク付けし、関連する主要なトレードオフについて説明します。 簡単に比較できるように、その結果をテーブルに要約します。 次のような要素を含めます。

    - 住宅市場と手頃さ
    - 生活費
    - 家族向け施設
    - 医療と安全性
    - ライフスタイルとコミュニティの特徴
    - 税金と財務上の考慮事項
    - ファーゴへの通勤時間

4.  要求に対するリサーチ ツールの分析を確認します。 問題がなければ、「**続けてください**」と入力します。

5.  結果を確認します。 次に、ファーゴとその周辺のコミュニティの半径 40 マイル以内の公立および私立高校に関するレポートを作成するようリサーチ ツールに依頼します。 使用可能な最新のデータを使用します。 すべての学校をランク付けし、すべての基準を順位付けで均等に扱います。 簡単に比較できるように、その結果をテーブルに要約します。 次のような要素を含めます。

    - 卓越したスポーツ プログラム
    - IB (国際バカロレア) および AP (アドバンスト プレイスメント) コースへのアクセス
    - 大学への進学実績
    - 課外活動の幅広さ
    - 学生と教師の比率
    - 卒業率

6.  要求に対するリサーチ ツールの分析を確認します。 問題がなければ、「**続けてください**」と入力します。

7.  結果を確認します。 次に、あなたは、最適なコミュニティと学校がどのように連携しているかを比較し、場所と教育の品質のトレードオフを特定したいと考えています。 コミュニティの分析の上位 5 位以内にランク付けされた都市と、学校の分析で上位 5 位以内にランク付けされた学校を比較するテーブルを作成するようにリサーチ ツールに依頼します。 次の列を含めます:

    - 都市名
    - 学校名
    - 都市のランキング
    - 学校のランキング
    - 主な特長 (手頃さ、施設、教育面の充実度)


8.  結果を確認します。 このタスクが示したように、リサーチ ツールはプロンプトが短くても取得でき、最新の Web データを使用して、徹底的で構造化された分析を提供できます。 この同じ機能を使用して、意思決定のサポートが必要なほぼすべてのトピックを調査、比較、評価することができます。
