# 演習 1、タスク 2: Excel で Copilot を使用してマネージャー概要レポートを生成する
---
タスク 1 で **Contoso_HR_ManagerMetrics.xlsx** スプレッドシートを分析し、エンゲージメント、離職率、トレーニングの傾向を特定しました。 次に、より詳細な分析を実行して、各マネージャーに関する実用的な分析情報を生成する必要があります。 この情報は、マネージャー概要レポートに取り込む必要があります。 Excel で Microsoft 365 Copilot を使用して、提供されたデータセットからマネージャーのメトリックを解釈し、主要な結果を要約し、マネージャー概要レポートで能力開発の機会を特定します。

このタスクを完了するには、次の手順を実行します。

1.  タスク 1 で更新した **Contoso_HR_ManagerMetrics.xlsx** ファイルは、引き続き **Excel for the web** で開いておく必要があります。 ファイルがまだ開いている場合は、**[アプリのスキル]** ペインの上部にある **[新しい会話を作成]** アイコンを選択して、新しいチャットを開始します。 前のタスクの最後にファイルを閉じた場合は、もう一度開いて、**[アプリのスキル]** ペインを開きます。

2.  必要に応じて、**[Sheet1]** を選択します。 **[アプリのスキル]** ペインで、マネージャーの業績データを分析し、改善のための推奨事項を提供するように Copilot に指示します。 業績が優れているマネージャーと、能力開発の機会があるマネージャーを特定します。 また、各マネージャーの上位 3 つの強みと改善の必要がある領域を要約するように Copilot に依頼します。 新しいシートに挿入できるテーブルに結果を表示します。

3.  結果を確認します。 そのテーブルを**マネージャーの業績**というタイトルの新しいシートに追加します。

4.  **[マネージャーの業績]** シートがまだ表示されていることを確認します。 チームの高い業績と最も強い相関関係があるメトリックを強調表示するように Copilot に依頼します。 新しいシートに挿入できるテーブルに結果を表示します。

5.  結果を確認します。 そのテーブルを**チームの業績**というタイトルの新しいシートに追加します。

6.  **[マネージャーの業績]** シートに戻ります。 トレーニングの完了率が低いことが離職率の上昇の一因になっている可能性があるマネージャーを特定して、トレーニングと離職率の間のパターンを強調表示するように Copilot に依頼します。

7.  結果を確認します。 そのテーブルを**トレーニングと離職率**というタイトルの新しいシートに追加します。

8.  **[マネージャーの業績]** シートに戻ります。 次に、タスク 3 でリーダーシップにメールで送信し、タスク 4 の計画セッションで使用できるように、このシートを PDF ファイルとしてエクスポートする必要があります。 このデータ シートをエクスポートするには、**[ファイル] > [エクスポート] > [PDF としてダウンロード]** (または **[PDF/XPS ドキュメントの作成]** と表示される場合もあります) の順に選択します。

    - お使いのバージョンの Excel で **[PDF としてダウンロード]** を選択した場合は、Copilot によって、Microsoft Edge ブラウザーの既存のタブに PDF が表示されます。 表示された **[エクスポート]** ペインで、**[ダウンロード]** ボタンを選択します。 Copilot によって、"マネージャーの業績" シートのテーブルを含む PDF ファイルがダウンロードされます。 このファイルは、お使いの PC の **ダウンロード** フォルダーに格納されます。 このファイルを**ダウンロード** フォルダーから OneDrive にコピーします。


    - お使いのバージョンの Excel で **[PDF/XPS ドキュメントの作成]** を選択した場合は、Copilot によって **[エクスプローラー]** ウィンドウが開かれます。 ファイルを OneDrive に保存します。
