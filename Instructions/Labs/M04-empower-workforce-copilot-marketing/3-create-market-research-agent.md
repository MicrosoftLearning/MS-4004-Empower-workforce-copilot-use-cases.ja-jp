# 演習 1、タスク 1: Copilot Studio を使って、市場調査エージェントを作成する
---
Relecloud は、商業オフィス市場で WorkSmart 360 を発売する準備をしています。 Relecloud のマーケティング部門がキャンペーンを通知するには、競争環境、顧客の好み、新たな傾向を理解することが重要です。 最初のステップは、Web ソース、ソーシャル メディア、調査レポートからこの情報を収集できる市場調査エージェントを作成することです。

このエージェントは、製品に依存しないものである必要があります。 Relecloud は最近 Microsoft 365 Copilot を導入したので、あなたは WorkSmart 360 の調査にそれを使う予定です。 ただし、マーケティング チームは将来的に、会社のさまざまな製品にそれを使うことを計画しています。

> [!NOTE]
> この演習では、Copilot Studio Lite エクスペリエンスを使って市場調査エージェントを作成します。 このシンプル エクスペリエンスは、普通のビジネス ユーザー向けに設計されており、プログラミング スキルは必要ありません。 一方、より複雑で高度なエージェントを構築するソフトウェア開発者は、通常、Copilot Studio のフル エクスペリエンスを使用します。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーで **Microsoft 365** のホーム ページ **(https://www.microsoft365.com)** にサインインします。

2.  Microsoft 365 のナビゲーション ウィンドウで **[新しいエージェント]** を選択します。 これにより、Copilot Studio の**エージェント ビルダー**が開き、**[新しいエージェント]** ページが表示されます。

3.  **[新しいエージェント]** ページでエージェントを作成するよう Copilot に指示します。 プロンプトには、エージェントの名前と、エージェントの概要、対象ユーザー、想定する動作などの一般的な説明を入力する必要があります。  
    <br/>このエージェントの場合は、次のプロンプトを入力し、進む矢印 (送信) アイコンを選択して、プロンプトを送信します。  
    <br/>**市場調査エージェントというタイトルのエージェントを作成します。このエージェントの目的は、要求された製品の市場分析情報を収集して分析することです。このエージェントは、ソーシャル メディア、調査レポート、顧客センチメント、競合他社の活動などの Web ソースから情報を集め、要求された製品について、傾向と機会を明らかにし、マーケティング戦略を示す必要があります。このエージェントは、マーケティング部門のメンバーによって使われます。**

4.  進む矢印を選択すると、新しいエージェント用の**エージェント ビルダー** フォームが表示されます。 このフォームの上部に、**[説明]** タブと **[構成]** タブがあります。

    - **[説明]** タブを使用して、Copilot と会話を続けることができます。 このタブは既定で表示されます。

    - **[構成]** タブでは、エージェントを動作させるための詳細な設定を定義できます。

    Copilot によってエージェントが作成されるまで 1 から 2 分待つと、**[エージェント プレビュー]** ペインにエージェントの名前と説明が表示されます。

5.  フォームの上部にある **[構成]** タブを選択します。 入力したプロンプトに基づいて Copilot が実行した内容を確認しましょう。

6.  **[構成]** タブの **[名前]** フィールドと **[説明]** フィールドは、入力したプロンプトに基づいて設定されているはずです。 **[指示]** フィールドまで下スクロールします。 最初のプロンプトで指定した説明に基づいて、Copilot によってこれらの指示が生成されました。 Copilot によって生成された指示の詳細レベルを確認します。

   > [!IMPORTANT]
   > エージェント ビルダー プロセスの優れた点は、基本的な自然言語による説明が、Copilot によって複雑な一連の指示に自動的に変換されることです。 このプロセスにより、ユーザーはこの詳細な指示セットを自分で作成しなくて済みます。

7.  指示を変更する場合は、**[指示]** フィールドで直接手動で編集するか、指示の更新を Copilot に依頼することができます。  
    <br/>**[指示]** を確認した後、まだ改良できるかもしれないとあなたは考えます。 どのように改善すればよいかわからないので、Copilot に意見を聞くことにします。  
    <br/>それには、**[説明]** タブを選びます。このエージェントを改善するために推奨される他の指示を Copilot に尋ねるプロンプトを入力します。

8.  Copilot の推奨を確認します。 その提案に満足したので、エージェントの指示にそれらをすべて追加するよう Copilot に依頼します。

9.  指示を更新したことを Copilot が応答したら、**[構成]** タブを選んで **[指示]** までスクロールします。 Copilot によって追加された新しい項目に注目してください。

10.  これで、指示は満足できるものになり、エージェントのナレッジ ソースとスターター プロンプトを構成する準備ができました。  
    <br/>**[構成]** タブで **[ナレッジ]** セクションまで下にスクロールし、**[すべての Web サイトを検索する]** トグル スイッチが有効になっていることを確認します。 Copilot は、元のプロンプトで指定された説明 (つまり、"… **などの Web ソースから収集し** …") に基づいてエージェントを作成したときに、このトグル スイッチを有効にしているはずです。 トグル スイッチが有効になっていない場合は、ここで有効にします。

11.  **[推奨プロンプト]** では、Copilot にプロンプトを生成させることも、独自のプロンプトを手作りすることもできます。 両方の方法を試してみましょう。  
    <br/>プロンプトの提案を Copilot に生成してもらうには、**[説明]** タブを選択し、エージェントに対して 3 つの提案されたプロンプトを生成するよう Copilot に依頼します。 各プロンプトにタイトルとメッセージがある点に注目してください。

12.  次に、独自のプロンプトをいくつか入力します。 **[構成]** タブを選択し、**[提案されたプロンプト]** セクションまで下にスクロールします。 Copilot によってエージェントに追加された 3 つのプロンプトが表示されます。  
    <br/>手作業で追加するプロンプトごとに、プロンプトの下に表示される **[推奨プロンプトを 1 つ追加する]** オプションを選びます。  
    <br/>一般的な市場調査アクションに関連する 6 つの推奨プロンプトが下に表示されます。 最初のプロンプト (**市場分析情報レポート**) を追加します。このプロンプトは次のタスクで使います。 次に、残りのプロンプトを確認し、他に気に入ったものを 2 つか 3 つ選んで、それもエージェントに追加します。
    
        - **タイトル:** 市場分析情報レポート (注: 次のタスクではこのプロンプトを使います。このプロンプトは必ず追加してください)。
            - **メッセージ:** {Product Category} に関する上位の傾向、対象者の行動、競争力のある開発、推奨されるマーケティング アクションを含むダウンロード可能な市場分析情報概要レポートを Microsoft Word で生成します。 カルチャの変化、コンテンツ消費パターン、インフルエンサーの役割を含めます。 これらの分析情報を使ってメッセージとチャネル戦略をどのように形成できるかを推奨します。

        - **タイトル:** ソーシャル メディア分析
            - **メッセージ:** {Product Name} に関する最近のソーシャル メディアの会話を分析します。 全体的なセンチメント、主要なテーマ、上位の問題点をまとめます。 マーケティング戦略に影響を与える可能性のある新しいトレンドや文化的シグナルを強調します。

        - **タイトル:** 競合他社の位置付け
            - **メッセージ:** {Product Category} に関する競合他社の精査を提供します。 主要な競合他社の位置付け、最近の製品発表、価格の変更、マーケティング キャンペーンを明らかにします。 自社の製品への影響をまとめ、カウンター戦略を提案します。

        - **タイトル:** 産業調査の統合
            - **メッセージ:** {Product Category} に関して公開されている業界レポートとホワイトペーパーをレビューします。 市場要因、阻害要因、予測の傾向をまとめます。 当社製品の戦略的機会とリスクを強調します。

        - **タイトル:** 発売キャンペーンのアイデア
            - **メッセージ:** 現在の市場センチメントと競合他社の活動に基づき、{Product Name} の発売キャンペーンに対して 3 つの実用的な推奨事項を提案します。 根拠、予想される影響、優先度のランク付け (P1 - P3) を含めます。

        - **タイトル:** 市場環境の概要
            - **メッセージ:** {Product Category} に関するダウンロード可能なエグゼクティブ向けの市場環境概要を作成します。 対象者のセンチメント、競合他社の位置付け、新たな傾向、マーケティング戦略に関する実用的な推奨事項を含めます。 わかりやすくするため、見出しと箇条書きを使って出力を構成します。
    
13.  提案されたプロンプトをいくつかテストします。 エージェントがデータをナレッジ ソースのドキュメントから正しく引き出していることを確認します。

14.  推奨プロンプトの結果に問題がなければ、**[作成]** ボタンを選んでエージェントを作成します。

15.  エージェントが作成されると、エージェントが正常に作成されたことを示すダイアログ ボックスが表示されます。 このダイアログ ボックスから、そのエージェントに移動することも、共有することもできます。 **[エージェントに移動する]** オプションを選択します。

> [!NOTE]
> この段階では、エージェントは非公開であり、アクセスできるのはあなただけです。 実務のシナリオで、エージェントを複数のチーム メンバーが使用する必要がある場合は、その人々と共有することになります。 このトレーニング演習では、あなた専用のテナント内で作業しているため、共有は必要ありません。
