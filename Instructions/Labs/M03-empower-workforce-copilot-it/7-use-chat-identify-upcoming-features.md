# 演習 2、タスク 1: Microsoft 365 Copilot Chat を使って Microsoft 365 の今後の機能を明らかにする
---
VanArsdel の次の IT リーダーシップ会議の準備として、あなたは次の四半期にユーザーに影響する可能性がある今後の Microsoft 365 の機能の概要を作成する必要があります。 IT チームがコミュニケーションとサポートの戦略を計画できるように、この情報はすぐに必要です。 目標は、今後 90 日以内に VanArsdel のエンド ユーザーに影響を与える可能性のある最も関連性の高い新機能を明らかにしてまとめ、IT 部門の対応の可能性に関する注意を示すことです。

これまでのこの作業では、チームは、ブログと管理センターの更新を検討し、結果を統合し、エンド ユーザーに対する各機能の影響を評価して、IT 部門での運用の効果を評価してから、プレゼンテーションのために最も重要な 8 つの項目の優先順位を付ける必要がありました。 ここでは、数時間かけてこの作業を終わらせる代わりに、Microsoft 365 Copilot Chat を使って今後の見通しを収集して要約する予定です。すべてを約 1 分で行い、Copilot を使うと時間が劇的に節約をされることを見ていきます。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーで **Microsoft 365** のホーム ページ **(https://www.microsoft365.com)** にサインインします。 このシナリオでは、Microsoft によって公開され、VanArsdel のテナントに格納されていない将来の機能を検索する必要があるため、**[作業]** オプションではなく **[Web]** オプションを選びます。

2.  今後の Microsoft 365 の機能を要約するよう Copilot に指示する次のプロンプトをコピーして貼り付けます。
  
   > [!NOTE]
   > この最初のプロンプトでは、概要ユニットで説明した 4 つの重要な要素が組み込まれている場合に、効果的なプロンプトがどのようになるかわかるよう、こちらでテキストを用意しました。 この演習の残りのプロンプトはすべて学習者が記述する必要がありますが、その際は、このプロンプトを手本として参考にできます。

   **私は VanArsdel, Ltd. の IT マネージャーです。今後 90 日以内に Microsoft によってリリースまたは一般提供される可能性が高い Microsoft 365 の機能のうち、エンド ユーザーに最も関連性の高いもの (コラボレーション、セキュリティ、Teams、Outlook、SharePoint、Viva など) をまとめています。各機能と共に、1 行の説明、技術部門以外のエンド ユーザーにとって重要な理由、IT 部門に推奨される対応 (コミュニケーション、トレーニング、テナント設定の変更など) を一覧にしてください。結果は短い箇条書きとして提供します (8 項目以下)。**

3.  Copilot の応答を検討します。 ソースが返された場合は、そのうちの 2 つを調べて、それらが有効であることを確認します。 あいまいな項目が返された場合は、チャットで明確化のためのフォローアップ質問をします。

4.  応答の最後にある **[応答のコピー]** アイコンを選びます。

5.  **Microsoft 365 Copilot Chat** のウィンドウで、ナビゲーション ウィンドウの **[アプリ]** アイコンを選びます。 表示される **[アプリ]** メニューで、**[Word]** を選びます。

6.  **Word for the web** で、空のドキュメントを作成してコピーした応答を貼り付けます。 Copilot Chat の応答をコピーして貼り付けると、通常、内容の先頭と末尾に余分なチャットの内容が含まれます。 コピーした応答を貼り付けた後、一緒に貼り付けられた余分なテキストを削除します。 コミュニケーションで優先する項目をタグで示すか強調表示にします。

7.  Word 文書で、**[ホーム]** タブの **[Copilot]** オプションを選びます。Copilot Chat ペインで、さらに 2 つの要求を Copilot に指示します。1) 箇条書きに基づいて経営陣向けの簡単な要約を記述し、2) 前に特定した各機能について、項目ごとに 1 文のリスクまたは機会を提供します。 文書の末尾にカーソルを置いてから、**[ドキュメントに追加] (+)** アイコンを選びます。

8.  後のタスクでこの文書を使用する予定なので、メニュー バーの上に表示されるファイル名を記録しておきます。
