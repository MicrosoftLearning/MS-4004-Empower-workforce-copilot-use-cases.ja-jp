# 演習 1、タスク 3:Copilot in PowerPoint を使用してエグゼクティブ プレゼンテーションを作成する
---
あなたは、Boulder の CIO から、重要なマイルストーン、リスク、次のステップを含めて、ネットワーク モダン化プロジェクトの状態に関するブリーフィングを発表するよう依頼されました。 これを行うため、あなたは Copilot in PowerPoint を使用して、プロジェクトの計画とリスク分析を、プロジェクトの現状、進捗状況、予想される結果を伝える、エグゼクティブ向けの明確なプレゼンテーションに変換したいと考えています。

#### Copilot in PowerPoint の使用

PowerPoint では、2 つの方法で Copilot を使用できます。スライド コンテンツや要約をすばやく生成するための標準の Copilot プロンプトと、スライド、レイアウト、プレゼンテーション構造に対してその場で直接編集を加えるための Copilot ペイン内の **[Copilot で編集]** です。 

- PowerPoint でスライドの下書きをすばやく作成する場合、内容を要約する場合、デッキの構成を変更せずに発表者ノートを作成する場合は、Copilot の標準プロンプトを使用することをお勧めします。 Copilot ペインを使用する場合、**[Copilot で編集]** を選択せずにプロンプトを入力すると、Copilot は、プレゼンテーションに直接インプレース変更を加えるのではなく、提案またはコンテンツを個別に生成するチャット スタイル モードで応答します。 

- スライドの再編成、スライドのテキストの修正、レイアウトの改善、複数のスライドにわたる反復的な編集など、プレゼンテーション内で Copilot による直接編集を利用する場合は、**[Copilot で編集]** を使用します。 **[Copilot で編集]** は、インプレース プレゼンテーション作業に最適化されているため、別の応答でコンテンツを提案するだけでなく、スライドの構造を理解してデッキに直接変更を適用できます。 

要約すると、アイデアを考え、生成するにはチャットスタイルの Copilot を使用し、ファイル内で実践的な編集を行うには、**[Copilot で編集]** を使用します。 通常、Copilot によってスライドやレイアウトの変更内容がプレビュー表示され、ユーザーの確認を受けて、スライド デッキに変更が直接反映されます。ユーザーがコピーと貼り付けを使って明示的に適用する必要はありません。

このタスクでは、**[Copilot で編集]** 機能を使用します。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーで、**Microsoft 365** のホーム ページにアクセスし、ナビゲーション ウィンドウで **[アプリ]** を選択し、**[アプリ]** メニューから **[PowerPoint]** を選択します。

2.  **PowerPoint for the web** で、空白のプレゼンテーションを作成します。

3.  **[ホーム]** タブのリボンが表示されない場合は、**[ホーム]** タブを選択します。 次に、**[ホーム]** タブ リボンの末尾にある **[Copilot]** を選択します。 

4.  Copilot ペインのプロンプト フィールドでプラス (+) 記号を選択し、ドロップダウン メニューから **[作業コンテンツの追加]** を選択します。 タスク 1 とタスク 2 で作成した Word ファイルを添付します。 

5.  プロンプト フィールドのプラス (+) 記号の横に **[Copilot で編集]** アイコンが表示されていることを確認します。 表示されていない場合は、プラス記号を選択し、ドロップダウン メニューから **[Copilot で編集]** を選択します。 プロンプト フィールドにアイコンが表示されます。 

6. ネットワーク モダン化プロジェクトの経営幹部向けプレゼンテーションを作成するよう Copilot に依頼します。 プレゼンテーションには、2 つの添付ファイルの主要な情報を要約する必要があります。 

7.  Copilot によってプレゼンテーションに関する一連の質問が表示されたら、適用する回答を選択します。 質問への回答が終わったら、**[確認]** ボタンを選択します。 また、一連の質問がさらに表示されることもあります。たとえば、スライド テンプレートを選択するよう求められることがあります。 テンプレートを選択しないと、Copilot は単に白地のスライドにテキストを配置するだけになることに注意してください。 繰り返しになりますが、適用する回答を選択するか、最適な判断を Copilot に任せる場合は **[すべてスキップ]** を選択します。

8.  Copilot in PowerPoint により、この情報に基づいてスライド一覧が生成されます。この処理には数分かかる場合があります。

9.  スライドを確認してから、あなたは 3 つのディスカッション スライドを追加することにします。 以下の各トピックについて、ディスカッション スライドを追加します (これらは、インフラストラクチャおよびモダン化プロジェクトについて経営幹部と CIO が持つ懸念事項の上位 3 つです)。

    - 運用とビジネス継続性のリスク
    - セキュリティ、コンプライアンス、データ保護
    - 予算、価値、配信の信頼度
    
    新しいスライドを依頼する前に、まずスライドをデッキ内のどこに表示するかを指定する必要があります。 この場合、これらのスライドをプレゼンテーションの末尾に追加する必要があります。 左側のスライド ペインで、スライドの一番下までスクロールし、最後のスライドの後を選択します。 

10. 「**ディスカッション: リスク、セキュリティ、価値**」というタイトルのトピック セクション スライドを追加するように Copilot に依頼するプロンプトを入力します。 このトピックには、1 枚目が「**Operational とビジネス継続性のリスク**」、2 枚目が「**セキュリティ、コンプライアンス、データ保護**」、最後に「**予算、価値、配信の信頼度**」という 3 枚のスライドが含まれている必要があります。 繰り返しになりますが、これらの新しいスライドが Copilot によって追加されるまでに 1 分から 2 分かかる場合があります。 

11.  スライド デッキの末尾まで下スクロールします。 最後の 4 枚のスライドには、新しく追加された「**ディスカッション: リスク、セキュリティ、価値**」トピックと、その 3 枚のディスカッション スライドを含める必要があります。 このトピックについて Copilot によって生成された各スライドを確認します。 各スライドに対して Copilot によって生成されたディスカッションのポイントは悪くはないものの、別の話題のポイントを追加することにしました。 以下のスライドにある既存の内容を次の話題に置き換えるよう Copilot に依頼します (数分かかる場合があります)。

        - **運用とビジネス継続性のリスク**
            - ダウンタイムやサービスの中断を予測していますか?
            - 堅牢な代替計画とロールバック計画はありますか?
            - シームレスな移行を実現し、顧客とスタッフへの影響を最小限に抑えにはどうすればよいですか?
    
        - **セキュリティ、コンプライアンス、データ保護**
            - テクノロジの変化によって生じた新たな脆弱性
            - 規制要件への準拠 (データ プライバシーや業界標準など)
            - インシデントを検出、対応、復旧する組織の能力
    
        - **予算、価値、配信の信頼度**
            - コスト超過と隠れた経費
            - 遅延と達成していないマイルストーン
            - 明確で測定可能な結果と ROI

12.  3 枚のスライドを確認し、Copilot が依頼どおりに変更を加えたことを確認します。 また、チャットの最後に表示された Copilot の提案も確認します。 気になる変更があれば、そのまま依頼できます。 

13. PowerPoint ファイルに満足したら、OneDrive アカウントに保存します。
