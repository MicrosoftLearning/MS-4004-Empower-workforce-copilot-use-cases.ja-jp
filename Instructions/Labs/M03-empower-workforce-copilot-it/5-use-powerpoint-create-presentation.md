# 演習 1、タスク 3:Copilot in PowerPoint を使用してエグゼクティブ プレゼンテーションを作成する
---
あなたは、Boulder の CIO から、重要なマイルストーン、リスク、次のステップを含めて、ネットワーク モダン化プロジェクトの状態に関するブリーフィングを発表するよう依頼されました。 これを行うため、あなたは Copilot in PowerPoint を使用して、プロジェクトの計画とリスク分析を、プロジェクトの現状、進捗状況、予想される結果を伝える、エグゼクティブ向けの明確なプレゼンテーションに変換したいと考えています。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーで、**Microsoft 365** のホーム ページにアクセスし、ナビゲーション ウィンドウで **[アプリ]** を選択し、**[アプリ]** メニューから **[PowerPoint]** を選択します。

2.  **PowerPoint for the web** で、空白のプレゼンテーションを作成します。

3.  空白のプレゼンテーションのタイトル スライドに表示される Copilot メニューで、**[ファイルを使用して新しいプレゼンテーションを作成]** オプションを選択します。

4.  **[Copilot でプレゼンテーションを作成する]** ウィンドウで、タスク 1 と 2 で作成した Word ファイルを添付します。 次に、Copilot にネットワーク モダン化プロジェクトのエグゼクティブ プレゼンテーションを作成するよう依頼します。 プレゼンテーションには、2 つの添付ファイルの主要な情報を要約する必要があります。

5.  Copilot in PowerPoint は、この情報を使ってプレゼンテーション トピックの一覧を生成します。 **[スライドの生成]** ボタンを選択するまで、スライドは生成されません。 代わりに、必要に応じてスライドを作成する前に変更できるよう、最初にトピックが表示されます。

6.  必要な場合は、トピックの順序を並べ替えることができます。 各トピック グループには、メイン トピックのタイトルとサブトピックの一覧が含まれます。 トピック領域内をポイントすると、トピックの左側に **[トピックの並べ替え]** アイコンが表示されます。 このアイコンを押したまま、トピック セクション全体を一覧内で上下にドラッグ アンド ドロップして、プレゼンテーション内の別の場所に移動します。

7.  プレゼンテーションからトピックを削除する場合は、トピックをポイントして、表示される **[削除]** (ごみ箱) アイコンを選びます。

8.  アウトラインに問題がなければ、プレゼンテーションに使用する **[プレゼンテーション スタイル]** を選択する必要があります。 必要に応じて、**[画像のユーザー設定]** を選択できます。 これで、Copilot でプレゼンテーションを生成する準備ができたので、**[スライドの生成]** ボタンを選択します。  

9.  Copilot でスライドが生成されるまでに 3 から 5 分かかる場合があります。 Copilot によるスライド プレゼンテーションの生成が完了すると、スライドが **[スライド一覧]** ビューに表示されます。 PowerPoint リボンで、**[表示]** を選びます。 **[表示]** リボンの **[プレゼンテーションの表示]** グループで、**[スライド一覧]** オプションが選ばれていることを確認します。 スライドを従来の順序で表示するには、**[標準]** を選びます。 これで、自由にスライドを確認し、必要な更新を行うことができます。 **デザイナー** ツールを使用すると、レイアウトを調整できます。 そのためには、**[ホーム]** タブを選択し、**[デザインの提案]** ボタンを選択します。このボタンは **[Copilot]** ボタンの左側に表示されます。

10. **[デザイナー]** ペインには、現在表示されているスライドに使用できる、さまざまなデザインのサンプルが表示されます。 プレゼンテーション内の別のスライドに変更し、そのスライドに対してどのような異なるデザイン オプションが生成されるかを **[デザイナー]** ペインで確認してください。 PowerPoint で現在のスライドを新しいデザインに置き換えるには、デザイン オプションのいずれかを選択します。 この機能は Copilot とは関係ありません。 ただし、これは Microsoft 365 サブスクリプションをお持ちの場合に PowerPoint でお使いいただける機能のため、以前に使用したことがない場合は、こちらで試してみてください。

11. 次に、Copilot が実行できる他の操作を見てみましょう。 Copilot in PowerPoint を使用するには、次の 2 つの方法があります。

    - 1 枚目のスライドの上に表示される **[Copilot]** アイコンを使用すると、開いているデッキを直接変更できます。 この Copilot の機能を使用して、スライドまたはトピック (複数のスライドを含むセクション) をプレゼンテーションに追加できます。

    - **[ホーム]** タブの端にある **[Copilot]** ボタンを選択すると、**[Copilot Chat]** ペインが開きます。 [Copilot Chat] ペインで Copilot とのチャットが開始されますが、通常、このペインで変更をスライド デッキに直接プッシュすることはできません。 代わりに、ユーザーが Copilot の応答をコピーし、貼り付ける必要があります。

    スライドを確認してから、あなたは 3 つのディスカッション スライドを追加することにします。 これらのスライドを、プレゼンテーション中にディスカッションを開始するためのプレースホルダーとして機能させたいため、ディスカッション トピック以外のテキストがスライドに表示されないようにします。 次のディスカッション トピックごとにスライドを追加します。

    - 運用とビジネス継続性のリスク
    - セキュリティ、コンプライアンス、データ保護
    - 予算、価値、配信の信頼度
  
    プレゼンテーションの完了後にこれらのディスカッションを行いたいので、スライド ペインでスライドの末尾まで下にスクロールし、最後のスライドの後を選択します。 最後のスライドの後にカーソルを置くと、Copilot に新しいスライドの挿入位置を示すことができます。

12. プレゼンテーションにスライドを追加するには、最初のスライドの上に表示される **[Copilot]** アイコンを選択します。 メニューで **[トピックの追加]** を選択します。

13.  表示される **[トピックの追加]** ウィンドウに、要求したトピックと各トピック内のスライドが表示されます。 最後のトピックが強調表示され、**[新しいトピックの追加] (+)** アイコンが表示されます。 **プラス記号** (+) を選択して、最後のトピックの後に新しいトピック セクションを挿入します。

        - **[追加するトピックの説明]** ウィンドウで、Copilot に「**ディカッション:リスク、セキュリティ、価値**」というトピック セクションを追加するよう依頼します。 このトピックには、1 枚目が「**Operational とビジネス継続性のリスク**」、2 枚目が「**セキュリティ、コンプライアンス、データ保護**」、最後に「**予算、価値、配信の信頼度**」という 3 枚のスライドが含まれている必要があります。
    
        - **[スライドの生成]** ボタンを選択します。

14.  Copilot でスライドが生成されると、スライドが **[スライド一覧]** ビューに表示されます。 メニュー バーの **[表示]** オプションを選択し、**[標準]** ビューを選択します。

15.  スライド デッキの末尾まで下スクロールします。 最後の 4 枚のスライドには、新しく追加された「**ディスカッション:リスク、セキュリティ、価値**」が含まれている必要があります。 このトピックの次の 3 つの各スライドについて、Copilot で生成されたコンテンツを、ディスカッションを促すために使用する予定の、次の話題に置き換えます。

        - **運用とビジネス継続性のリスク**。 テキストのセクションを次の質問に置き換えます。
            - ダウンタイムやサービスの中断を予測していますか?
            - 堅牢な代替計画とロールバック計画はありますか?
            - シームレスな移行を実現し、顧客とスタッフへの影響を最小限に抑えにはどうすればよいですか?
    
        - **セキュリティ、コンプライアンス、データ保護**。 テキストの既存のセクションを次の話題に置き換えます。
            - テクノロジの変化によって生じた新たな脆弱性
            - 規制要件への準拠 (データ プライバシーや業界標準など)
            - インシデントを検出、対応、復旧する組織の能力
    
        - **予算、価値、配信の信頼度**。 テキストの既存のセクションを次の話題に置き換えます。
            - コスト超過と隠れた経費
            - 遅延と達成していないマイルストーン
            - 明確で測定可能な結果と ROI

16.  PowerPoint ファイルをお使いの OneDrive アカウントに保存します。
