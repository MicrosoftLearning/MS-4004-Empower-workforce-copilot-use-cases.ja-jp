# 演習 2:Microsoft 365 Copilot を使用して機能のリリースと導入を変革する
---
Microsoft 365 Copilot は、IT 担当者が組織全体でテクノロジの導入を計画し、伝達し、測定する方法を変革しています。 IT チームにとって、導入の成功は新しいツールを展開することだけではなく、従業員がそれらを理解し、受け入れ、積極的に使用できるようにすることです。 IT 担当者は、Microsoft 365 の今後の機能の迅速な認識、エンド ユーザー向けの明確で魅力的な伝達事項の作成、お知らせ、ベスト プラクティス ガイド、トレーニング資料などの実用的なリソースの開発に Copilot を活用すると、そのプロセスのすべてのステップを合理化できます。

> [!TIP]
> このモジュールの「概要」ユニットでは、効果的なプロンプトの次の 4 つの重要な要素について説明しました。目標、コンテキスト、ソース、期待値。 この演習でプロンプトを作成する際は、これらの要素に留意してください。

### シナリオ

あなたは、3 つの地域で事業を展開し、2,500 人の従業員を擁するエンジニアリングおよび製造会社である VanArsdel Ltd. の IT マネージャーとして、組織の生産性、つながり、セキュリティを維持する上で重要な役割を果たします。 VanArsdel のリーダーシップは最近、Microsoft 365 機能の導入を迅速化するための全社的な取り組みを発表しました。 従業員が新しいツールにアクセスできるだけでなく、効果的に使用してコラボレーションを改善し、毎日の作業を効率化する方法を理解することが望まれています。

同社は、AI を活用した会議の概要から、改善されファイル コラボレーションとセキュリティ更新プログラムにいたるまで、次の四半期にいくつかの新しい Microsoft 365 機能をロールアウトする準備をしています。 しかし、VanArsel での過去のリリースは、技術的な準備だけでは十分ではないことを示しました。 同社は、明確なコミュニケーション、教育、積極的なフィードバック ループがなければ、導入が行き詰まり、ユーザーの不満が高まるということを身をもって学びました。

過去の失敗を繰り返さないようにするため、CIO は、導入を促進し、混乱を最小限に抑えるように設計された "90 日間の機能認識キャンペーン" を主導するよう IT 部門に指示しました。 このキャンペーンでは、明確なコミュニケーション、データドリブンな分析情報、従業員が自信を持って新しいツールを採用できるようにするユーザー フレンドリなリソースを組み合わせる必要があります。 あなたの目標は、Microsoft 365 Copilot を使用して、この取り組みの効果を計画、伝達、測定し、以前は何週間もかかった手動の作業を AI の支援により合理化された作業の時間に変えることです。

この演習では、次の Microsoft 365 Copilot 機能を、調査から分析までキャンペーンを導くために使用する予定です。

- Copilot Chat を使って VanArsdel ユーザーに最も関連性の高い今後の Microsoft 365 機能の上位 8 つを調査して要約する。 年

- Copilot in Viva Engage で、従業員が質問、アイデア、フィードバックを共有できる Viva コミュニティと共に、新機能に対する認識と興奮を高める魅力的なアナウンス投稿を作成する。

- Copilot Chat で、従業員が 1 日目から新しい機能を自信を持って使用できるように、明確でわかりやすく実践的なアドバイスを提供する「10 のベスト プラクティスのヒント」クイック ガイドを生成する。

- アンケート エージェントで、早期導入の進行状況を追跡し、阻害要因を特定し、最初のロールアウト フェーズでユーザーのセンチメントを理解するためのフィードバック アンケートを設計する。

- アナリスト エージェントで、実際のアンケート結果を調べ、導入傾向を視覚化し、CIO 向けのエグゼクティブ分析情報を生成する。


この演習では、計画とコミュニケーションからフィードバックと分析にいたるまで、すべて Microsoft 365 Copilot を利用してデータに基づく完全な機能導入キャンペーンを作成します。 あなたは、IT 担当者が複雑なロールアウト プロセスを明確で魅力的で測定可能な成功事例に変えるのに、Copilot がどのように役立つかを直接体験します。
