# 演習 2、タスク 2:Copilot in PowerPoint を使用してエグゼクティブ プレゼンテーションを作成する
---
次のタスクは、明日の財務リーダーシップ会議で、**スマート センサーの契約比較**ドキュメントで取り込んだ情報を共有することです。 視覚的なわかりやすさと簡潔さが重要です。 あなたは、Copilot in PowerPoint (Office エージェント) を使用して、重要なポイントを要約し、特定されたリスクを軽減するための提案を含むプレゼンテーションを生成する予定です。 デッキが簡潔で視覚的に魅力的であることを確認します。

#### Copilot in PowerPoint の使用

PowerPoint では、2 つの方法で Copilot を使用できます。スライド コンテンツや要約をすばやく生成するための標準の Copilot プロンプトと、スライド、レイアウト、プレゼンテーション構造に対して直接インプレース編集を加えるための Copilot ペイン内の **[Copilot で編集]** です。

- PowerPoint でスライドの下書きをすばやく作成する場合、内容を要約する場合、デッキの構成を変更せずに発表者ノートを作成する場合は、Copilot の標準プロンプトを使用することをお勧めします。 Copilot ペインを使用する場合、**[Copilot で編集]** を選択せずにプロンプトを入力すると、Copilot は、プレゼンテーションに直接インプレース変更を加えるのではなく、提案またはコンテンツを個別に生成するチャット スタイル モードで応答します。  
    
- スライドの再編成、スライドのテキストの修正、レイアウトの改善、複数のスライドにわたる反復的な編集など、プレゼンテーション内で Copilot による直接編集を利用する場合は、**[Copilot で編集]** を使用します。 **[Copilot で編集]** は、インプレース プレゼンテーション作業に最適化されているため、別の応答でコンテンツを提案するだけでなく、スライドの構造を理解してデッキに直接変更を適用できます。

要約すると、アイデアを考え、生成するにはチャットスタイルの Copilot を使用し、ファイル内で実践的な編集を行うには、**[Copilot で編集]** を使用します。 通常、Copilot によってスライドやレイアウトの変更内容がプレビュー表示され、ユーザーの確認を受けて、スライド デッキに変更が直接反映されます。ユーザーがコピーと貼り付けを使って明示的に適用する必要はありません。

このタスクでは、**[Copilot で編集]** 機能を使用します。

このタスクを完了するには、次の手順を実行します。

1.  Microsoft Edge ブラウザーで、**Microsoft 365** のホーム ページにアクセスし、ナビゲーション ウィンドウで **[アプリ]** を選択し、**[アプリ]** メニューから **[PowerPoint]** を選択します。

2.  **PowerPoint for the web** で、空白のプレゼンテーションを作成します。

3.  **[ホーム]** タブのリボンが表示されない場合は、**[ホーム]** タブを選択します。 次に、**[ホーム]** タブ リボンの末尾にある **[Copilot]** を選択します。

4.  Copilot ペインのプロンプト フィールドでプラス (+) 記号を選択し、ドロップダウン メニューから **[作業コンテンツの追加]** を選択します。 前のタスクで作成した **Smart Sensor contract comparison** ファイルを添付します。

5.  プロンプト フィールドのプラス (+) 記号の横に **[Copilot で編集]** アイコンが表示されていることを確認します。 表示されていない場合は、プラス記号を選択し、ドロップダウン メニューから **[Copilot で編集]** を選択します。 プロンプト フィールドにアイコンが表示されます。

6.  Copilot ペインのプロンプト フィールドに、Copilot に作成を依頼する内容の説明を入力します。 効果的なプロンプトの 4 つの重要な領域に対処することを忘れないでください。 添付した **Smart Sensor contract comparison** ファイルの主要な調査結果を要約したスライド プレゼンテーションを作成するよう Copilot に依頼する必要があります。 プレゼンテーションは、財務リーダーシップ チームを対象とする必要があります。 これには、価格、配信タイムラインと使用条件、保証とサポートなど、主要な調査結果のトピック領域ごとに個別のスライドを含める必要があります。 各スライドでは、選択した領域の Adatum Corporation と Contoso, Ltd. の契約内容を比較する必要があります。 プレゼンテーションを魅力的なものにするためにビジュアルを使用し、わかりやすくするためにスライドに箇条書きを使用するように Copilot に指示します。

7.  Copilot によってプレゼンテーションに関する一連の質問が表示されたら、適用する回答を選択します。 質問への回答が終わったら、**[確認]** ボタンを選択します。 また、一連の質問がさらに表示されることもあります。たとえば、スライド テンプレートを選択するよう求められることがあります。 テンプレートを選択しないと、Copilot は単に白地のスライドにテキストを配置するだけになることに注意してください。 繰り返しになりますが、適用する回答を選択するか、最適な判断を Copilot に任せる場合は **[すべてスキップ]** を選択します。

8.  Copilot in PowerPoint により、この情報に基づいてスライド一覧が生成されます。この処理には数分かかる場合があります。

9.  Microsoft のテストでは、スライド生成に関して結果は一定ではありませんでした。 追加の確認を求められず、Copilot によってスライドが自動生成されることもありました。 また、Copilot Chat ペインにスライドのアウトラインが表示され、今後の進め方について選択肢がいくつか提案されたこともありました。 後者のシナリオになった場合は、アウトラインに基づいて処理を進めるように指示します。 いずれの場合も、通常、Copilot によるスライド生成には数分かかりました。

10. Copilot で生成されたスライドを確認します。 財務部門のリーダーシップ向けプレゼンテーションの基礎はしっかりしているものの、変更したい点がいくつかあるようです。 まず、リスク軽減策についてリーダーシップから質問されるのは確実ですが、プレゼンテーションではその点について触れられていません。 新しいスライドを依頼する前に、まずデッキのどこにスライドを表示するかを指定する必要があります。 スライドを確認し、このトピックが適している場所を見つけます。 次に、左側のスライド ペインで、Copilot によるスライドの挿入場所を選択します。 場所を選択したら、リスク軽減のための主要な推奨事項が記載されたスライドを追加するよう Copilot に求めるプロンプトを入力します。 スライドには、重要なポイントを強調するために、吹き出しや強調表示などの視覚的な工夫も加えるようにします。

11. Copilot から、この新しいスライドに関連する一連の質問が再び聞かれる場合があります。 実際、最初のスライド生成時にテンプレートを指定していたとしても、使用するテンプレートを選択するよう求められる場合があります。 Copilot によるスライド生成には数分かかる場合があります。

12. Microsoft のテストでも、追加の確認なしで Copilot によってスライドがスライド デッキに自動的に追加されたことがありました。 また、[Copilot Chat] ペインにスライドのアウトラインが表示され、今後の進め方について選択肢がいくつか提案されたこともありました。 後者のシナリオが発生した場合は、予定していたスライドをプレゼンテーションに追加するよう指示してください。 いずれの場合も、通常、Copilot によるスライド生成には数分かかりました。
    
13. 次は、「ベンダーの利点の要約」スライドを追加するよう Copilot に依頼します。 このスライドはデッキの末尾に追加するので、デッキの適切な場所にカーソルを移動します。 次に、プレゼンテーションで取り上げた各トピックについて、より優れた点があるのはどのベンダーかを示す「ベンダーの利点の要約」スライドを追加するよう Copilot に依頼します。

14. Copilot によってデッキの末尾に生成されたスライドを確認します。 最後に、Copilot でアクション プラン スライドを作成します。 これをデッキの最後のスライドにするので、スライド デッキの最後のスライドの後を選択します。 次に、ネゴシエーションとリスク軽減のための次の手順を示す「アクション プラン」スライドを追加するよう Copilot に依頼します。 スライドには、タイムラインと担当者の役割が含まれている必要があります。

15. Copilot で生成されたアクション プラン スライドを確認します。 テスト段階では、Copilot がタイムラインを含められなかったことがありました。 同様のことが起きた場合は、次のステップのタイムラインを作成するように Copilot に依頼する新しいスライドを追加できます。 Copilot でタイムライン スライドが生成されたら、個別のスライドのままにするか、タイムラインを元のスライドに手動でコピーできます。

16. お使いの OneDrive アカウントに、PowerPoint ファイルを **Smart Sensor contract comparison - Finance** として保存します。
