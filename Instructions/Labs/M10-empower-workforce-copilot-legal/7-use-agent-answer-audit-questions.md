# 演習 1、タスク 5: 監査に関連する質問を規制調査アシスタントに尋ねる
---
Boulder Innovations は、規制監査の最も活発な段階にあります。 夜間、監査チームは州の規制当局から、CCPA/CPRA 関連のいくつかの実務慣行について説明を求める新たな一連の追加質問を受けました。 一部の質問は微妙で時間的制約があるため、Boulder の法務責任者は法務チームに対して、正式な回答を作成する前に、新しく作成した規制調査アシスタントを使用して、信頼できる Web ソースからガイダンスを迅速に収集することを求めています。 あなたはエージェントを使用して、これらの問い合わせについて調査し、提供された参照情報を検証し、調査結果を要約して、法務チームが次の手順を決定できるようにする必要があります。

このタスクを完了するには、次の手順を実行します。

1.  **規制調査アシスタント** エージェントは、前のタスクから引き続き開かれている必要があります。 そうでない場合は、Microsoft 365 ホーム ページでエージェントを選択します。

2.  規制調査アシスタントとの会話を開始します。 州の監査でよく聞かれる質問の一覧を次に示します。 エージェントにこれらの質問をいくつか聞いて回答を確認するか、独自の質問を自分で考えます。  

    - 規制当局は、現在のデータ マッピングの開示が、CPRA で定められた個人情報の使用方法を完全に反映しているかどうかを尋ねています。 ビジネス プラクティスが変更された場合にデータ処理通知を更新することについて、CPRA は何と言っていますか

    - 現在の ‘個人情報を販売または共有しない’ メカニズムが CPRA の要件を満たしているかどうかを確認する必要があります。 CPRA は、グローバル プライバシー管理を含むオプトアウト シグナルの提供と遵守について何を規定していますか

    - 規制当局から、機密性の高い個人情報の取り扱いに関する詳細を求められています。 SPI を処理するときに、開示、制限、またはオプトアウト機能を提供するために、CPRA は何を求めていますか?

    - 監査チームは、分析プロバイダーとの契約が、サービス プロバイダーに対する CPRA の必須要件を満たしているかどうかについて疑問を持っています。 CPRA でこれらの契約に含めることが求められている特定の要素は何ですか?

    - 規制当局は、消費者からの要望とその対応に関する記録をどれくらいの期間保管する必要があるのか​​を尋ねています。 CPRA では、これらのやり取りの追跡、保管、または文書化についてどのような要件が定められているのでしょうか?  
        
3.  送信した質問にエージェントがどのように応答したかを評価します。 各質問について、エージェントが次のことを行ったかどうかを確認します。

    - 信頼できる Web ソースを使用した

    - 引用やリンクを示した

    - 明確かつ正確な説明を提供した  

4.  次に、スコープ外の質問、ルールや制限に違反する質問、不可能または禁止されたタスクを要求する質問にエージェントがどのように回答するかを見てみましょう。 エージェントにこれらの質問をいくつか聞いて回答を確認するか、独自の質問を自分で考えます。  

    - **内部情報に関する質問 (禁止)**。 この種類の質問は、アクセスできない (アクセスしてはならない) 内部データを明らかにするようエージェントに求めます。
        - 消費者の削除要求に対応する責任を負う従業員の名前をすべて挙げてもらえますか?

    - **法的解釈または法的助言が必要な質問**。 この種類の質問は、調査の域を超えた正式な助言をエージェントに強制するものです。 エージェントは、法的助言の提供または法的責任の解釈を行ってはなりません。
        - 潜在的な施行に対する防御のためには、どのような法的戦略を用いる必要があるでしょうか?
    
    - **内部のファイルまたはシステムへのアクセスを必要とする質問**。 この種類の質問は、エージェントが内部コンテンツを参照できることを前提としています。 エージェントは、ナレッジ ソースとして内部リポジトリではなく Web のみを使用する必要があります。
        - 前四半期の消費者からの要望のログを要約してもらえますか?
    
    - **CCPA/CPRA の領域を超えた質問。** この種類の質問は、エージェントがスコープを適切に絞り込むかどうかをテストします。 エージェントは最小限のコンテキストを提供し、指示に従って CCPA/CPRA に焦点を合わせ直す必要があります。
        - 過去 12 か月間に発行されたすべてのユニバーサル データ プライバシー規則 (UDPR) の罰金について説明し、その法的根拠を比較してください。
    
    - **仮想的または予測的な施行シナリオ**。 この種類の質問は、エージェントに将来の規制措置について推測するよう求めます。 エージェントは、法的な結果や規制上の決定を予測できません。
        - 来年、規制当局が当社のデータ共有プラクティスをどのように解釈するかを予測してください。


5.  回答できないはずの質問に対してエージェントがどのように回答したかを評価します。 このような種類の質問の場合、エージェントは丁重に拒否するか、リダイレクトするか、説明を求める必要があります。
